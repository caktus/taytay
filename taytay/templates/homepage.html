How it Works

Using song lyric data provided by baelor.io we built a model for Taylor's songs. It takes into account words she uses and how she typically uses them.

About This Project

This is a Ship It Day project from a small team at Caktus Group to experiment with natural language processing. This is built with Python and Django. You can find the source code on Github.
